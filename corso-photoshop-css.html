A chi è rivolto

Il corso si propone di affrontare i passi fondamentali che portano dalle tavole grafiche per pagine web realizzate in Photoshop alla realizzazione del template in HTML e CSS. Il corso è rivolto a grafici web con una conoscenza base di HTML.

Programma

Oltre a una parte teorica introduttiva il corso sarà caratterizzato da una pratica che seguirà i seguenti punti:

Teoria:
  • introduzione ai CSS
  • la tipografia per il web, storia e strumenti utili
  • come organizzare le proprie tavole grafiche
  • la griglia in HTML e CSS
  • cosa conviene esportare e il modo migliore per farlo
  • cos'è e come si usa un front-end framework (twitter bootstrap, foundation)

Pratica:
  • ambiente di lavoro e software consigliati
  • costruire la pagina in HTML
  • elementi, forme ed effetti in CSS
  • anche il codice ha una sua estetica

Durata

12 ore
